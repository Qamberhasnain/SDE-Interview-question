# Rotate Matrix by One Element
### Problem Statement:
Given a 2-dimensional matrix of size ‘N’ x ‘M’, rotate the elements of the matrix clockwise.

#### Example:
```
Input Matrix: [ [ 1, 2, 3 ] 
                [ 4, 5, 6 ] 
                [ 7, 8, 9 ] ]

Output Matrix: [ [ 4, 1, 2 ] 
                 [ 7, 5, 3 ] 
                 [ 8, 9, 6 ] ]

The output matrix is generated by rotating the elements of the input matrix in a clockwise direction.
Note that every element is rotated only once.
```
# Solution:

```java
public static void rotateMatrix(ArrayList<ArrayList<Integer>> mat, int n, int m) {
		
		int row = 0, col = 0;
		int prev, curr;
		while(row<n && col<m){
			if(row + 1 == n || col + 1==m) break;
			prev = mat.get(row+1).get(col);
			for(int i=col;i<m;i++){
				curr = mat.get(row).get(i);
				mat.get(row).set(i,prev);
				prev = curr;
			}
			row++;
			for(int i=row;i<n;i++){
				curr = mat.get(i).get(m-1);
				mat.get(i).set(m-1,prev);
				prev = curr;
			}
			m--;
			if (row < n){
				for(int i = m-1; i >= col; i--){
					curr = mat.get(n-1).get(i);
					mat.get(n-1).set(i,prev);
				   	prev = curr;
				}
			}
			n--;
			 if (col < m){
				for(int i = n-1; i >= row; i--){
					curr = mat.get(i).get(col);
					mat.get(i).set(col,prev);
					prev = curr;
				}
			 }
			col++;
		}
	}
```
